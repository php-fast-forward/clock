Coverage and Reports
====================

The repository publishes the following external reports:

- `Coverage Report <https://php-fast-forward.github.io/clock/coverage/index.html>`_
- `Testdox Report <https://php-fast-forward.github.io/clock/coverage/testdox.html>`_

Those workflows delegate execution and report generation to the shared Fast Forward development tooling.

If your repository publishes coverage or TestDox artifacts to GitHub Pages or another static host, this is the place to link
them. When working locally, the quickest source of truth remains the PHPUnit command line output and the generated coverage
artifacts from the Fast Forward tooling.

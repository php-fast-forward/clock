# Examples

This directory contains examples demonstrating how to use FastForward Clock.

## Usage Examples

1. **`01-system-clock.php`** - Basic SystemClock usage
2. **`02-system-clock-with-timezone.php`** - SystemClock with specific timezone
3. **`03-frozen-clock.php`** - Basic FrozenClock usage
4. **`04-frozen-clock-inputs.php`** - FrozenClock with different input formats
5. **`05-frozen-clock-container.php`** - Using FrozenClock with Fast Forward Container
6. **`06-psr20-clock-interface.php`** - Resolving clock via PSR-20 interface

Run examples with:

```bash
php examples/01-system-clock.php
```
